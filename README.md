# 🏥 Al-Hayat Hospital Healthcare Analytics Dashboard

An end-to-end **Healthcare Analytics Dashboard** built with **Power BI** to analyze hospital operations, patient performance, appointments, financial performance, and doctor productivity through interactive visualizations and KPIs.

---

## 📌 Project Overview

This project demonstrates the complete Business Intelligence workflow:

- Data Cleaning & Transformation using Power Query
- Data Modeling
- DAX Measures & Calculated Columns
- Interactive Dashboard Design
- Business Performance Analysis

---

## 🛠️ Tools & Technologies

- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📊 Dashboard Features

- Executive Overview
- Patient Analysis
- Doctor Performance
- Appointment Analysis
- Financial Dashboard
- Forecast & What-If Analysis

---

## 📈 Key Performance Indicators (KPIs)

- 👨‍⚕️ Total Patients
- 🩺 Total Doctors
- 📅 Total Appointments
- 💰 Total Revenue
- 📊 Collection Rate
- 💵 Average Revenue per Patient
